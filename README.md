# 🏛️ Culture Project: Armenian Heritage Interactive

Этот проект — интерактивный образовательный портал, посвященный великим армянским деятелям и культурным памятникам. 

### 🚀 Технологии:
* **Frontend:** HTML5, CSS3 (адаптивный дизайн), JavaScript.
* **Backend:** Python (Flask).
* **AI Интеграция:** * **Groq API** — для генерации умных ответов и диалогов.
  * **ElevenLabs** — для реалистичной озвучки персонажей.

### 🌟 Особенности:
* **Интерактивные страницы:** Узнайте историю Гарегина Нжде, Тиграна Великого и других героев.
* **Голос истории:** Благодаря ElevenLabs, персонажи могут «говорить» с пользователем.
* **Умный помощник:** Встроенный ИИ-чат для ответов на вопросы о культуре.
